# 天気予報アプリ

スマートフォン対応の天気予報アプリです。現在の天気、5日間の予報、服装アドバイス、傘の必要性を表示します。

## 機能

- 🌤️ 現在の天気情報
- 📅 5日間の天気予報  
- 👕 気温に応じた服装アドバイス
- ☂️ 傘の必要性判定
- 📱 モバイル対応（PWA）
- 🔍 都市名検索

## デモ

https://[あなたのユーザー名].github.io/weather-app/

## 使用技術

- HTML/CSS/JavaScript
- OpenWeatherMap API
- Progressive Web App (PWA)

## セットアップ

1. このリポジトリをクローン
2. OpenWeatherMap APIキーを取得
3. `js/app.js`の1行目にAPIキーを設定
4. `index.html`をブラウザで開く
